Origem da demanda:

A Porto Seguro envia documentos(papel) de:
• Guia De Honorário Individual
• Guia de Consulta Médica
• Guia de Serviço Profissional/Serviço Auxiliar de Diagnóstico e Terapia

A JRM digitaliza os documentos listados a cima para serem validado.
Iremos desenvolver um sistema para validação dos documentos digitalizados, onde os operadores iram preencher os campos do formulário de acordo com o documento digitalizado.
Após preenchimento, o formulário será enviado no formato XML para o WebService da Porto Seguro para processar as informações. Feito isso o WebService retorna o formulário XML informando Sucesso ou Erro(informa erro).

• Caso Sucesso, o documento saíra da fila de validação e as informações salvas no banco de dados.
• Caso Erro, o documento volta para validação informando o erro. O operador devera corrigir e aprovar novamente o formulário que será enviado novamente para o WebService da Porto Seguro.
